Front End For  MYSQL

 

A B O U T

The Project Front End for MYSQL is the front end to the open source database server MYSQL. It aims at providing a complete front end to MYSQL and thus reducing the difficulty caused while interacting the system using query language SQL. It also aims at providing a complete tool for DBA to assist in the database management tasks. It thus eases the job of DBA. The System has been developed taking the ideas into account.

The system has been developed using Java technology. So, This ensures the portability of the system. It uses JDBC bridge for connecting with the MySQL server. The IDE used was Netbeans. The software development process used was Unified Process. UML was being used as common modelling tool during initial phases of system development.
